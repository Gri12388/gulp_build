# Простая сборка Gulp

С помощью этих файлов можно быстро настроить сборку проекта на Gulp

## Инструкция

1. Скачать все файлы в любую дирректорию
2. Ввести в терминал команду npm i
3. Выполнить команду Gulp
4. Начать писать код.
